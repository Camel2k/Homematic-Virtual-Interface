Coordinator Kommandos

 Kommando Funktion createmesh|player1,player2... erzeugt eine Gruppe aus player1 und player2 .... standalone|player1 entfernt player1 aus der Gruppe toggle|player1 schaltet player1 ein oder aus (wenn der Player in einer Gruppe ist wird er entfernt) switchon|player1 schaltet player1 ein (wenn ein anderer Player Musik spielt wird player1 mit diesem gruppiert) switchoff|player1 schaltet player1 aus playFav|player1|FavName Spielt den Sonso Favoriten 'FavName' auf player1 ab. (Wenn FavName 'random' ist wird ein zufälliger Favorit gespielt) volume|player1|volume setzt die Lautstäke von player1 settransportstream|player1|TSName setzt den transportstream auf player1 (TSName muss RINCON..... sein) mehrere Kommandos können * getrennt angegeben werden. (Bitte keine Leerzeichen für Player oder fav Namen verwenden)
