Roomba980 Plugin

Dieses Plugin generiert eine 4 Tasten Fernbedienung in der CCU. Das Gerät heisst dort Cleo980.
Nachdem die Zugangsdaten zum Roboter in den Einstellungen dieses Plugins gesetzt worden, kann der Roboter über die 4er FB gesteuert werden.

Kanal 1 : Kurzer Tastendruck = Start des Roboters
Kanal 2 : Kurzer Tastendruck = Stop des Roboters
Kanal 3 : Kurzer Tastendruck = Docking des Roboters

Kanal 4 hat 3 Datenfelder, die regelmässig aktualisiert werden.
Feld : CUR_STATE = Aktueller Status des Roboters
Feld : BATTERIE = Aktueller Batteriestand
Feld : BIN = true wenn Staubbehälter voll ist


Das Aktualisierungsintervall liegt normal bei 2 Minuten. Dies kann in den Plugin Einstellungen verändert werden.
